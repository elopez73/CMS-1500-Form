Segments

This segment prints out the Last Name of the Insurance Policy Holder. If the insurance type is "Medicare" or "Medicare Part A", as specified in the EDI/Eligibility tab in the Insurance Edit window, then this segment will print out up to 60 characters for the Insured's Last Name. If the insurance type is anything other than "Medicare" or "Medicare Part A", then this segment will print out the Insured's Last Name. It will also do the following to special characters: Replace "'" with "" Replace "'" with "" Replace """ with "" Replace "~" with "" Replace "*" with "" Replace ":" with "" Replace "-" with ""
This segment prints out the First Name of the Insurance Policy Holder. If the insurance type is "Medicare" or "Medicare Part A", as specified in the EDI/Eligibility tab in the Insurance Edit window, then this segment will print out up to 35 characters for the Insured's First Name. If the insurance type is anything other than "Medicare" or "Medicare Part A", then this segment will print out the Insured's First Name. It will also do the following to special characters: Replace "'" with "" Replace "'" with "" Replace """ with "" Replace "~" with "" Replace "*" with "" Replace ":" with "" Replace "-" with ""
